## 📦 About

A replica of Amazon's ADPTA conveyor belt system that simulates package flow and tracks items through the fulfillment process. Built as part of a class project to model real-world warehouse logistics.

---

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/Oluwa-Temmy/434-fullfillment-center.git
cd 434-fullfillment-center

# Install dependencies
pip install -r requirements.txt

# Run the app
flask run
```

---

## ⚙️ Features

- Conveyor belt simulation modeled after Amazon's ADPTA system
- Real-time package tracking and routing
- Web interface to visualize and interact with the simulation
- Backend logic powered by Python and C++
